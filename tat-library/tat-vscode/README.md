# TAT Language Support

Local VS Code syntax highlighting for TryAngleTree `.tat` files.

## Development

Open this folder in VS Code and press F5 to launch an Extension Development Host.

To launch from a terminal:

```sh
code --new-window --extensionDevelopmentPath="/Users/cjohanson/Desktop/TryAngleTree Stuff/Language V1/tat-code-reviewer/tat-vscode"
```
